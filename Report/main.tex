\begin{abstract}
Side-channel attacks targeting programs which are executed on Intel SGX processors are of common vulnerabilities in this platform. The security of code and data in the cloud is provided by execution of the program in the SGX enclave, but it is subject to side-channel attacks which target the enclave and retrieve data by observing page-faults, cache-misses, TLB misses, etc. In this report, we review and discuss common attacks on SGX enclaves and the protection which is needed and currently trending in research. In our study, we have observed and analyzed several types of research done in this area and compared them with each other.% Another importance in this area is the performance loss caused by protection techniques which we also discuss in this report.
\end{abstract}
